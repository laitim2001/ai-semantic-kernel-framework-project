## 測試步驟詳情

### 1.1 查看系統設定

```yaml
測試步驟:
  1. 導航到 /settings 頁面
  2. 確認設定頁面正確載入

預期結果:
  - 顯示當前系統設定
  - 敏感資訊 (API Key) 已遮罩
```

### 1.3 API Key 管理

```yaml
測試步驟:
  1. 導航到設定頁面
  2. 找到 API Key 配置區域
  3. 輸入/更新 API Key
  4. 保存

預期結果:
  - API Key 安全保存
  - 不在 UI 中完整顯示 Key
  - 可測試 Key 有效性
```

### 1.4 連接測試

```yaml
測試步驟:
  1. 導航到設定頁面
  2. 點擊 "Test Connection" 按鈕

預期結果:
  - 顯示測試結果
  - 成功時顯示綠色確認
  - 失敗時顯示錯誤原因
```
