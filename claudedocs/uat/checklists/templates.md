## 測試步驟詳情

### 1.1 模板列表顯示

```yaml
測試步驟:
  1. 導航到 /templates 頁面
  2. 確認列表正確載入

預期結果:
  - 顯示可用的 Workflow 模板
  - 每個模板顯示名稱、描述、分類
```

### 2.2 從模板創建 Workflow

```yaml
前置條件:
  - 有可用的模板

測試步驟:
  1. 選擇一個模板
  2. 點擊 "Use Template" 按鈕
  3. 輸入新 Workflow 名稱
  4. 確認創建

預期結果:
  - 成功創建新 Workflow
  - Workflow 結構與模板一致
  - 導航到新 Workflow 編輯器
```
