## 測試步驟詳情

### 1.1 總 Agents 數量顯示

```yaml
前置條件:
  - 系統至少有 1 個 Agent

測試步驟:
  1. 打開 Dashboard 頁面
  2. 查看 "Agents" 統計卡片
  3. 確認數字與實際 Agent 數量一致

預期結果:
  - 顯示正確的 Agent 數量
  - 數字格式正確 (千位分隔)
  - 載入過程有 loading 狀態

驗證 API:
  GET /api/v1/dashboard/stats
  預期返回: { "agents_count": N, ... }
```

### 1.2 總 Workflows 數量顯示

```yaml
前置條件:
  - 系統至少有 1 個 Workflow

測試步驟:
  1. 打開 Dashboard 頁面
  2. 查看 "Workflows" 統計卡片
  3. 確認數字與實際 Workflow 數量一致

預期結果:
  - 顯示正確的 Workflow 數量
  - 點擊可導航到 Workflows 頁面
```

### 1.3 活躍執行數量顯示

```yaml
測試步驟:
  1. 打開 Dashboard 頁面
  2. 查看 "Active Executions" 統計卡片
  3. (可選) 啟動一個 Workflow 執行

預期結果:
  - 顯示當前活躍執行數量
  - 有執行時數字大於 0
  - 無執行時顯示 0
```

### 1.4 成功率百分比顯示

```yaml
測試步驟:
  1. 打開 Dashboard 頁面
  2. 查看 "Success Rate" 統計卡片

預期結果:
  - 顯示百分比格式 (如 95.5%)
  - 無執行歷史時顯示 N/A 或 0%
  - 計算準確 (成功數/總執行數)
```

### 3.1 快速創建 Agent 按鈕

```yaml
測試步驟:
  1. 打開 Dashboard 頁面
  2. 找到 "Create Agent" 或類似按鈕
  3. 點擊按鈕

預期結果:
  - 按鈕可見且可點擊
  - 點擊後導航到 /agents/new 或打開創建對話框
  - 無錯誤發生
```
