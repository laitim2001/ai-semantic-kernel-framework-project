## User Stories

### U1: Expert List Page
**作為**管理員，**我希望**能在 UI 頁面看到所有 Agent Expert 定義，**以便**了解系統中有哪些可用專家。

### U2: Create Expert Page
**作為**管理員，**我希望**能透過表單建立新的 Agent Expert，**以便**擴充專家團隊。

### U3: Edit Expert Page
**作為**管理員，**我希望**能修改現有 Expert 的配置，**以便**調整 prompt、工具等設定。

### U4: Expert Detail Page
**作為**管理員，**我希望**能查看 Expert 的完整配置，**以便**了解其能力範圍。
