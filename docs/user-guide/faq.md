## 帳戶與登錄

### Q: 如何重置密碼？

1. 在登錄頁面點擊「忘記密碼」
2. 輸入註冊時使用的電子郵件
3. 檢查收件箱（包括垃圾郵件文件夾）
4. 點擊重置鏈接
5. 設置新密碼

### Q: 密碼有什麼要求？

密碼必須符合以下要求：
- 至少 12 個字符
- 包含大寫字母
- 包含小寫字母
- 包含數字
- 包含特殊字符 (!@#$%^&*)

### Q: 如何更改電子郵件地址？

請聯繫系統管理員更改電子郵件地址，因為電子郵件是帳戶的主要識別符。

### Q: 帳戶被鎖定怎麼辦？

連續 5 次登錄失敗後，帳戶會被鎖定 15 分鐘。您可以：
- 等待 15 分鐘後重試
- 聯繫管理員解鎖帳戶

## 工作流相關

### Q: 如何創建工作流？

1. 導航到「Workflows」頁面
2. 點擊「創建工作流」
3. 填寫名稱和描述
4. 設計流程圖
5. 保存工作流

詳細指南請參考 [工作流管理指南](./workflows.md)

### Q: 工作流可以有多少個節點？

工作流最多可以有 50 個節點，包括：
- 開始和結束節點
- Agent 節點
- 審批節點
- 條件分支節點
- 並行執行節點

### Q: 如何複製現有工作流？

1. 在工作流詳情頁面
2. 點擊「更多操作」(三點圖標)
3. 選擇「複製工作流」
4. 修改名稱
5. 保存

### Q: 工作流執行失敗怎麼辦？

1. 查看執行詳情中的錯誤信息
2. 檢查審計日誌了解詳細錯誤
3. 常見原因：
   - Agent 配置錯誤
   - API 連接問題
   - 輸入數據格式不正確
4. 修正問題後重新執行

### Q: 如何設置定時執行？

1. 進入工作流詳情
2. 點擊「觸發器」標籤
3. 添加「定時觸發」
4. 設置 Cron 表達式或選擇預設時間
5. 保存觸發器

常用 Cron 表達式：
- `0 9 * * *` - 每天早上 9 點
- `0 9 * * 1-5` - 工作日早上 9 點
- `0 0 1 * *` - 每月 1 號

## Agent 相關

### Q: Agent 和工作流有什麼區別？

| 項目 | Agent | 工作流 |
|------|-------|--------|
| 定義 | AI 助手，執行特定任務 | 任務執行順序的定義 |
| 功能 | 處理輸入、生成輸出 | 協調多個 Agent 的執行 |
| 使用 | 被工作流調用 | 調用 Agent |

### Q: 如何讓 Agent 使用外部工具？

1. 編輯 Agent 配置
2. 在「工具」部分啟用所需工具
3. 配置工具參數（如 API 密鑰）
4. 在指令中說明何時使用工具
5. 測試 Agent 行為

### Q: Agent 回應太慢怎麼辦？

可能的解決方案：
1. 減少 max_tokens 參數
2. 簡化 Agent 指令
3. 使用更快的模型（如 gpt-3.5-turbo）
4. 檢查工具調用是否過多

### Q: 如何使用模板創建 Agent？

1. 導航到「Templates」頁面
2. 瀏覽可用模板
3. 選擇合適的模板
4. 點擊「使用此模板」
5. 根據需要自定義
6. 保存為新 Agent

## 審批流程

### Q: 如何審批待處理項目？

1. 點擊導航欄的「Approvals」
2. 查看待審批列表
3. 點擊項目查看詳情
4. 選擇「批准」或「拒絕」
5. 添加評論（可選）
6. 確認決定

### Q: 可以批量審批嗎？

是的：
1. 勾選多個待審批項目
2. 點擊「批量操作」
3. 選擇「全部批准」或「全部拒絕」
4. 確認操作

### Q: 審批有時間限制嗎？

是的，默認設置：
- 標準審批：24 小時
- 緊急審批：4 小時
- 超時後會自動升級給管理員

### Q: 如何委託審批權限？

1. 進入「設置」→「審批委託」
2. 選擇委託人
3. 設置委託時間範圍
4. 選擇可委託的工作流類型
5. 保存設置

## 整合與觸發器

### Q: 如何配置 Webhook 觸發？

1. 進入工作流詳情
2. 點擊「觸發器」標籤
3. 添加「Webhook 觸發」
4. 複製生成的 Webhook URL
5. 在外部系統（如 n8n）配置此 URL

Webhook URL 格式：
```
https://ipa-platform.azurewebsites.net/api/v1/triggers/webhook/{workflow_id}
```

### Q: 支持哪些外部系統整合？

目前支持：
- ServiceNow
- Dynamics 365
- n8n
- Teams（通知）
- 自定義 Webhook

### Q: 如何配置 Teams 通知？

1. 在 Teams 中創建 Incoming Webhook
2. 複製 Webhook URL
3. 在「設置」→「整合」中配置 Teams
4. 粘貼 Webhook URL
5. 選擇通知類型
6. 保存設置

## 故障排除

### Q: 頁面加載緩慢怎麼辦？

嘗試以下步驟：
1. 刷新頁面 (Ctrl+F5)
2. 清除瀏覽器緩存
3. 檢查網絡連接
4. 嘗試不同的瀏覽器
5. 如持續問題，聯繫技術支持

### Q: 看不到某些功能？

可能原因：
1. 權限不足 - 聯繫管理員
2. 功能未啟用 - 檢查訂閱計劃
3. 瀏覽器兼容性 - 使用 Chrome/Edge

### Q: 數據似乎不正確？

1. 刷新頁面獲取最新數據
2. 檢查時間範圍篩選
3. 清除瀏覽器緩存
4. 如果問題持續，聯繫支持

### Q: 如何報告問題？

1. 收集信息：
   - 錯誤消息截圖
   - 發生時間
   - 操作步驟
2. 聯繫支持：
   - 郵件: support@ipa-platform.com
   - 工單系統: https://support.ipa-platform.com
