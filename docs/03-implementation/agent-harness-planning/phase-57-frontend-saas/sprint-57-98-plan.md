**Purpose**: The harness-deepening proposal (`claudedocs/1-planning/harness-deepening-proposal-20260610.md` ┬¦1, workflow A) identified that verification today is an **outer wrapper** (`verification/correction_loop.py`'s `run_with_verification()`) around `AgentLoop.run()`, not a gear of the main loop. Because `resume()` drives the **same** `_run_turns` (the 57.90 re-entrancy payoff), the in-loop gate **automatically covers the resume path** ŌĆö closing the structural hole for free.
