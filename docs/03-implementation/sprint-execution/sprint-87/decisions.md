## 詳細說明

### D87-1: UI 組件庫選擇

**背景**: DevUI 需要快速搭建專業的開發者工具界面

**選項分析**:
1. Ant Design - 現有項目已使用，功能完整
2. Material UI - 需要額外學習，風格與現有不一致
3. 自定義組件 - 開發成本高

**決策**: 選擇 Ant Design

**原因**:
- 與現有前端保持技術棧一致
- 提供表格、分頁、過濾等現成組件
- 團隊已熟悉使用方式

---

### D87-2: 路由設計

**背景**: DevUI 是開發者專用工具，需要獨立的路由空間

**決策**: 路由掛載在 `/devui` 下

**路由結構**:
```
/devui                    → DevUI 首頁/追蹤列表
/devui/traces             → 追蹤列表
/devui/traces/:id         → 追蹤詳情
```

**原因**:
- 與主應用 `/app` 路由分離
- 便於後續添加更多開發者工具
- 可以單獨控制訪問權限

---

### D87-3: API 狀態管理

**背景**: DevUI 需要頻繁調用 API 獲取追蹤數據

**決策**: 使用 React Query (TanStack Query)

**原因**:
- 自動緩存和去重
- 內建加載和錯誤狀態
- 支援分頁和無限滾動
- 自動重試失敗請求
