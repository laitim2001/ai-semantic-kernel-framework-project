- **影響**: 中 - 可能造成狀態不一致
- **計劃**: 使用 Write-Through 策略

- **描述**: 長對話的訊息查詢可能變慢
- **影響**: 低 - 初期數據量不大
- **計劃**: 實現 cursor-based pagination

- **影響**: 中 - 影響儲存和頻寬
- **計劃**: 配置化大小限制，預設 10MB
