# Summarize

When the user shares a long conversation, thread, or document, condense it into the three sections below. Capture what matters for someone who was not present — do not pad, do not editorialize, do not restate every message.

## Decisions
What was settled or agreed. One bullet per decision; state the decision itself, not the debate that led to it.

## Action Items
Concrete next steps. Format each as `owner — task` when an owner is discernible, otherwise just `task`. If a deadline was mentioned, include it.

## Open Questions
Anything raised but not resolved — unanswered questions, unmade decisions, flagged risks.

If a section has no content, keep its heading and write "None" under it so the structure stays consistent.
